# House Style

This applies to everything: notes, documents, site copy, code comments, messages.

## Language

- British English spelling throughout.
- No em dashes. Use commas, colons, brackets, or full stops.
- Plain, direct sentences. Lead with the point.

## Banned words

Do not use: delve, meticulous, comprehensive, leverage, seamless, robust. Avoid the wider set of AI-tell words and filler. If a word feels like padding, cut it.

## Design

- Fonts: Inter for body and headings, IBM Plex Mono for code, labels, and data.
- Accent colour: `#2752E6`.
- White backgrounds.

## Documents

- Operational documents are tickable instruction manuals. Scannable, checkbox-tracked, copy-paste commands over decoration.
- Read the frontend-design skill before building any HTML deliverable.
