# Attribution

Parts of this brain adapt ideas and skill patterns from the open-source project **claude-obsidian** by AgriciDaniel (https://github.com/AgriciDaniel/claude-obsidian), which implements the LLM Wiki pattern.

Adapted into this vault:

- the immutable-source (`.raw/`) plus generated-notes split
- the hot / index / area read hierarchy for retrieval
- the query, lint and save skill patterns

Not used: the desktop Claude Code plugin machinery (shell and Python scripts, Makefile, git hooks, file-locking, transport detection, verifier agent, methodology-mode scripts, multi-tool configs). This brain runs mobile and chat-first through the GitHub connector, not from a desktop terminal, so that machinery would not execute.

Their project is released under its own licence. See the source repository.
