## arXiv watch

cs.AI and cs.MA RSS both returned their Monday 13 July build today (a day late relative to the calendar, the same lag pattern flagged in recent reports), so this is the first chance to pull from it. Five agentic/multi-agent picks worth flagging:

- **GATS: Graph-Augmented Tree Search with Layered World Models for Efficient Agent Planning** (arxiv.org/abs/2607.08894) — combines tree search with a layered world-model representation to make long-horizon agent planning cheaper without losing plan quality.
- **Long-Horizon-Terminal-Bench** (arxiv.org/abs/2607.08964) — a new benchmark specifically testing agents on long-horizon terminal/CLI tasks with dense reward-based grading, relevant to anyone evaluating coding-agent reliability over multi-step sessions.
- **TrustX Agent Risk Classification Framework (ARC)** (arxiv.org/abs/2607.09586) — a risk-tiering framework for internally deployed agentic AI systems, aimed at enterprises that need to classify which agents can act unsupervised.
- **Agora: Enhancing LLM Agent Reasoning Via Auction-Based Task Allocation** (arxiv.org/abs/2607.09600) — proposes an auction mechanism for allocating subtasks across a multi-agent team, an alternative to fixed role assignment.
- **Shared Selective Persistent Memory for Agentic LLM Systems** (arxiv.org/abs/2607.09493) — a shared-memory architecture letting agent teams selectively persist and retrieve context across sessions, adjacent to the OpenWiki Brains shared-memory work LangChain published 10 July.
