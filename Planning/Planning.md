# Planning

Week plans and time-boxed planning docs. Back to [[CLAUDE]].

## How it works

- One dated file per week plan: `Week Plan YYYY-MM-DD.md` (dated to the Monday).
- Each plan grounds in the current repo and chat state before it is written, so it reflects where things actually are.
- Open loops carry forward from one week to the next.
