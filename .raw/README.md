# .raw

Original source documents. Pasted transcripts, exports, briefs, long reports, anything I want kept exactly as received.

Rule: Claude reads these but never edits them. They are the immutable record. Any note Claude generates from a source goes into an area or project and links back here.

This is different from `_inbox/`, which is transient capture to be sorted and cleared. `.raw/` is kept.

Pattern adapted from the LLM Wiki / claude-obsidian project. See ATTRIBUTION.md.
