# Skill: weekly review

Run on request, or weekly via routine.

## Steps

1. Read the week's daily notes in `00-dashboard/daily/`.
2. For each area, summarise what moved and what stalled.
3. List open loops across all areas, ranked by priority.
4. Flag anything stale (no update in 30 or more days).
5. Clear any `#needs-filing` items still in `_inbox/`.
6. Write the review to `00-dashboard/daily/YYYY-MM-DD-weekly-review.md`.
