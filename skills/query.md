# Skill: Query

Answer a question from the vault.

## Steps

1. Read [[Hot]] first for recent context.
2. If that is not enough, read [[Dashboard]], then the relevant area note.
3. Only then open individual notes.
4. Answer from what is in the vault. If the vault does not hold it, say so plainly rather than guessing.
5. Name the notes you used.

## Rules

- Do not read the whole vault for a simple question.
- Lead with the answer.
