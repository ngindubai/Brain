# Skill: New Project

Run when I start a new piece of time-boxed work.

## Steps

1. Make `Projects/<Name>/` and create the tracker `Projects/<Name>/<Name>.md` from [[Project Template]]. Name the tracker after the project, in Title Case, so the graph reads clearly.
2. Interview me for the one goal if I have not stated it. One sentence only.
3. Fill the tracker: the one goal, your role, status, with status, stage and next_action in the frontmatter.
4. Link the project from the relevant area note (for example [[Products]]).
5. Add it to [[Projects Board]] and [[Open Tasks]].
6. If promoting from `Research/`, copy the research note in as the basis, link back, and mark the research note promoted.
