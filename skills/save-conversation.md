# Skill: save conversation

File the current conversation as a structured note.

## Steps

1. Summarise the conversation: the point, the decisions, the open loops.
2. Decide the destination area or project per the filing rule.
3. Write a clean note, named and dated, with frontmatter and `[[wikilinks]]`.
4. If the conversation produced raw material worth keeping verbatim, save that to `.raw/` untouched and link to it.
5. Update `00-dashboard/hot.md` with a one-line entry.
6. Log it in today's daily note.
