# Skill: new project

Run when I start a new piece of time-boxed work.

## Steps

1. Make `projects/[name]/` and create the tracker `projects/[name]/[name].md` from the template in `projects/_template/plan.md`. Name the tracker after the project, not `plan`, so the graph reads clearly.
2. Interview me for the one goal if I have not stated it. One sentence only.
3. Fill the tracker: the one goal, your role, status, with status, stage and next_action in the frontmatter.
4. Link the project from the relevant area note (for example [[areas/products/products]]).
5. Confirm where inputs should land.
6. If promoting from `research/`, copy the research note in as the basis, link back to it, and mark the research note promoted.
