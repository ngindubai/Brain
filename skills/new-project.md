# Skill: new project

Run when I start a new piece of time-boxed work.

## Steps

1. Copy `projects/_template/` to `projects/[name]/`.
2. Interview me for the one goal if I have not stated it. One sentence only.
3. Fill the project CLAUDE.md: goal, role, status.
4. Link the project from the relevant area `_index.md`.
5. Confirm where inputs should land.
