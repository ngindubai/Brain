# _inbox

Raw capture lands here. Anything I jot on mobile that I have not filed, drop it in this folder.

Claude and the daily routine read this folder, classify each note, move it to the right area or project, name it, link it, and clear it. If something cannot be placed confidently, it stays here with a `#needs-filing` tag.

Capture is mostly chat-first, so this folder is the fallback for when I am in Obsidian and not talking to Claude.
